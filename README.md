
 <h1 align="center"> 📊 Student Grade Manager - Python</h1>
 
 Projeto acadêmico desenvolvido em Python com o 
 objetivo de praticar conceitos básicos da linguagem.
O sistema realiza o cadastro de alunos, cálculo de médias 
e análise das estatísticas da turma.
Foi desenvolvido como parte do processo de aprendizagem e 
consolidação dos conhecimentos fundamentais em Python.



  # 📌 Descrição do Projeto
  Este projeto acadêmico consiste em um sistema simples de cadastro e cálculo de médias de alunos, desenvolvido em Python. 
  O programa permite inserir a quantidade de estudantes, registrar seus nomes e notas, calcular automaticamente a média individual 
  e apresentar os resultados de forma organizada.



 #▶️ Exemplo de Execução

```
How many students do you want add? (From 2 to 7): 2

--------STUDENT 1--------
Enter the student's name: Maria
Enter the first grade (0 to 10): 8
Enter the second grade (0 to 10): 9

--------STUDENT 2--------
Enter the student's name: João
Enter the first grade (0 to 10): 6
Enter the second grade (0 to 10): 7

======== STUDENT AVERAGE ========

Grade 1: 8
Grade 2: 9
Average: 8.5

Name: João
Grade 1: 6

Average: 6.5

======== CLASS SUMMARY ========

The class average is: 7.50

The student with the lowest average was João with 6.50
```



# 📊 Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| Cadastro de alunos | Permite inserir múltiplos estudantes |
| Cálculo de médias | Calcula média individual automaticamente |
| Média da turma | Calcula média geral da classe |
| Maior média | Identifica o melhor desempenho |
| Menor média | Identifica o menor desempenho |
